## ✨ 기술적인 도전

1. Github actions + Codedeploy를 사용하여 자동화 배포 적용

2. (2차 배포) PostgreSQL의 ORDBMS 특징을 활용하여 DB에 List형태로 데이터를 삽입하여 table의 row 수 감축

3. (2차 배포) Apache Kapka를 통한 멤버 초대시 알림 Push 기능 구현
